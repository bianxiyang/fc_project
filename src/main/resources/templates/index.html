API 接口信息

用户管理

GET /api/fc-users - 获取所有用户列表
GET /api/fc-users/{id} - 根据ID获取用户信息
GET /api/fc-users/name/{name} - 根据名称获取用户
POST /api/fc-users - 创建新用户
PUT /api/fc-users/{id} - 更新用户信息
DELETE /api/fc-users/{id} - 删除用户

赛程管理

POST /api/matches/generate-schedule - 生成赛程
GET /api/matches - 获取所有比赛
GET /api/matches/{id} - 获取特定比赛
PUT /api/matches/{id} - 更新比赛信息（如比分）
